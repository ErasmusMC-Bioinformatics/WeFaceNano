Nanopore Analysis Pipeline

Basecalling, de-barcoding and FASTQ conversion

Albacore is used to read the raw FAST5 files and will do another run of basecalling with de-barcoding
and will convert the reads to FASTQ.
When FASTQ files are being used Albacore will not be used.

Assembly

Canu or Miniasm are used for assembly.
The file created by Canu or Miniasm containing the contigs will be used for further analysis.

Circularize contigs

Circlator 1.5.5 can be used to try and circularize the contigs.
This option can be selected in the pipeline settings menu and is set to OFF by default.
Running the circularize option can take a long time.
SPAdes 3.7.1 will be used for reassembling by Circlator.

Plasmid Detection

BLAST 2.7.1 is used for plasmid detection.
BLAST can be used with a remote database (NT or NR) or with a local NT or toxoplasma database.

Antibiotic Resistance

Resfinder is used to find antimicrobial resistance genes based on the contig file created by Canu or Miniasm.

View Results

Antibiotic Resistance and BLAST

The results will be presented in different headers (Assembly, Resfinder, BLAST and Plasmids).
The Assembly header will show the contigs created by Canu or Miniasm and the length of these contigs.
The Resfinder header will show the antibiotic resistance genese found per contig. It will also show the percentage identity of the found gene.
